# /pilot-evidence

Daily and per-encounter evidence captured during the pilot.

Suggested structure (created on demand):

```
/pilot-evidence/
  <YYYY-MM-DD>-smoke.json                # Daily 11-check smoke output
  <YYYY-MM-DD>-evidence-review.md        # Legal reviewer sample notes
  <YYYY-MM-DD>-scenarios-summary.md      # QA scenario run summary
  feedback/
    physician/
    patient/
    legal/
    compliance/
  incidents/
    <ticket-id>/
```

This folder is the canonical home for all pilot-window operational evidence. Do not delete artifacts; archive on close of pilot.
